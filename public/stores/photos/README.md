# In-store display photos

Drop photos of Eco Global Foods products on store shelves here.

- Accepted formats: .jpg .jpeg .png .webp
- They appear automatically on the /stores ("Stores available at") page,
  in an "On the shelves" gallery, sorted by filename.
- Suggested size: landscape, ~1200x900px.

Example: public/stores/photos/al-fatah-shelf-1.jpg

Note: on Vercel the filesystem is ephemeral, so photos committed to the repo
persist, but photos uploaded at runtime do not. Commit them to the repo (or move
to cloud storage) for production.
